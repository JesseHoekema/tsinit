# TSinit

A quick script to just make a typescript project that just work

# How to run?

## For Mac/Linux:

enter: `curl -fsSL https://raw.githubusercontent.com/JesseHoekema/tsinit/refs/heads/main/setup.sh | bash` in your terminal, make sure you run this in your project directory

## For Windows:

1. open powershell
2. enter: `irm https://raw.githubusercontent.com/JesseHoekema/tsinit/refs/heads/main/setup.ps1 | iex`
